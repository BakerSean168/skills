# Roadmap

## Current state

The repository currently contains:

- a coaching-first learning skill
- a repository-specific skill scaffold tool
- shared repo conventions in `AGENTS.md`

## Near-term

- add more learning-oriented coaching skills for specific stacks
- improve `coding-coach` session patterns and map maintenance rules
- extend `skill-scaffold` with optional richer templates when justified
- add lightweight validation helpers for local skill consistency

## Mid-term

- add domain-specific skills such as frontend coaching, TypeScript coaching, or architecture review
- standardize richer examples for high-value skills
- improve discoverability across skills with tighter descriptions and naming

## Non-goals for now

- building a large plugin framework around this repo
- adding heavy automation before the skill set stabilizes
- generating many low-value skills just to increase count
